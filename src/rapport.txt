Description de l'utilisation :
	
	Utilisation de la commande java pour le lancement de l'executable .jar avec l'option -jar,
	avec en options propre au programme le nom du fichier (image) au format pgm à traiter suivi
	du nom du fichier dans lequel enregistrer le résultat sous forme :
	
	java -jar ModelisationReduction.jar [fichierSource.pgm] [fichierDestination.pgm]

	Exemple :
	java -jar ModelisationReduction.jar super.pgm super_reduit.pgm


Modifications apportées :

    - Utilisation de la fonction d'énergie avant pour la production du graphe directement depuis l'image.
    - Suppression de lignes en plus des colonnes.
    - Gestion complète des images .ppm en couleur (P3).
